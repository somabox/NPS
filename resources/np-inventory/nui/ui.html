Ajuda do inventário:

Para mover todos os itens, Certifique-se de que a QUANTIDADE esteja vazia
Para alterar a QUANTIDADE, use a ROLAGEM DO MOUSE. Você pode fazer isso enquanto arrasta ou usa os itens.
Para USAR um item, arraste-o para o icone USAR.
A QUANTIDADE não será redefinida ao Fechar o inventario, então se acostume a alterar as quantidades!

Keybinds (defaults):

  • Shift + Clique: Moves half of the stack to opposite inventory.
  • Ctrl + Click: Moves amount inputted in AMOUNT to opposite inventory.
  • Left Click: Normal Drag (configurable to be like Right Click in settings)
  • Right Click: No Hold to Drag
  • Middle Click: Uses Item
  • Scroll Wheel: While hovering over AMOUNT will increase/decrease value
  • Numbers 0 - 9: Will input the typed number.
  • Any other key: Will set AMOUNT to blank/move entire stack.
  • Pressing Ctrl + C while hoving over an item will copy the description, useful for getting evidence numbers.

Lojas:

Coloque a QUANTIDADE o valor que você deseja comprar e arraste o item da loja para o seu inventário.

Crafting:

Verifique se você possui todos os itens necessários para craftar seu novo item.
Ao craftar, você tem que colocar a quantidade exata de itens necessários.

Durabilidade dos items:

Quase todos os itens tem durabilidade (incluindo armas). Durabilidade na cor vermelha indica que o item esta destruido.
